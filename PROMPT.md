# Промпт для промо-сайта MiMo Code Telegram Bot

Скопируй это в новый чат MiMo и получи готовый сайт:

---

Сделай одностраничный промо-сайт для Telegram Mini App бота "MiMo Code". Это AI-ассистент для удалённого управления ПК через Telegram.

Функции бота:
- Общение с AI (MiMo Code) прямо из Telegram
- Мониторинг ПК в реальном времени (CPU, RAM, GPU, VRAM)
- Запуск терминала и выполнение команд на удалённом ПК
- Загрузка файлов и фото (drag & drop)
- Работает 24/7 даже после перезагрузки ПК (автозапуск)

Дизайн:
- Тёмная тема, стильный минимализм как у современных AI-продуктов
- Шрифт Space Grotesk
- Цвета: тёмный фон #0c0b0e, акцент #b8a9ff (приглушённый фиолетовый), зелёный #7ee8a0
- Секции: Hero с анимированным заголовком, Features (4 карточки), Как работает (3 шага), CTA кнопка "Открыть бота"
- Hero: крупный заголовок "Управляй ПК из Telegram", подзаголовок, кнопка → https://t.me/mimo1ai_bot
- Features: Чат с AI, Мониторинг ПК, Терминал, Файлы — каждая карточка с иконкой и описанием
- Как работает: 1) Нажми /start 2) Открой Mini App 3) Управляй удалённо
- Фон: subtle animated gradient или floating particles
- Адаптивный (мобилка + десктоп)
- Один HTML файл, всё инлайн (CSS + JS), без зависимостей кроме Google Fonts

Код должен быть самодостаточным — один .html файл который можно открыть в браузере.
